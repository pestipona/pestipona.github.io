Creating A Callback Function

November 15, 2022 by Paul

In JavaScript I learned something new called Callback Functions. This is a function that is passed into another function as an argument and invoked to perform a specific task. I've seen callbacks applied in other programming languages like C++ and Python.

One way to illustrate the concept of callbacks is to show it as an example used in code.


    // Data list of numbers defined in an array.
    const list = [5, 8, 9, 2, 7, 6, 3, 1, 4];

    // Callback function creates a list of numbers < 5:
    function callback(list){
        const newList = [];
        for (let i = 0; i < list.length; i++) {
            if (list[i] < 5){
                newList.push(list[i]);
            }
        } return newList;
    }

    // Filter function using a callback to create a list of numbers < 5:
    function filter(list, callback) {
        return callback(list);
    }

As can be seen in the previous example above the Callback Function was passed into the Filter Function and applied to each element of the data list which was an array. This is a very useful feature that has many applications in JavaScript.

Why Use A Style Guide?

December 31, 2022 by Paul

Using a CSS Style Guide greatly enhances the quality of one's webpage. As I'm starting to learn how to design and create web pages. I'm starting to appreciate the value of CSS Style Guides. One can start from scratch which is always the ideal way to learn the basics and core concepts of CSS such as "Grids". But sometimes the situation dictates on what style to use or even the use of pre-made styles and templates like those offered in Bootstrap to create quick and efficient designs. One of the reasons why I would use a CSS framework such as Bootstrap are the following:

  • Quick way to design a webpage when there is no time.
  • Great resource to design a webpage quickly.
  • Tremendous amount of styling offered.
  • Gives more time to focus on the functionality of the website than the design.

So overall Bootstrap is a great design tool, and has many advantages but one should use this with caution. It is not an ideal way to start learning CSS. The developer is always encouraged to learn first the basics of CSS and then learn to use Bootstrap.
